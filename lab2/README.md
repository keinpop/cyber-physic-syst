## Что делает скрипт

`inference.py` запускает 10 запросов к локальному Ollama-серверу с моделью
`qwen2.5:0.5b` и сохраняет результаты в файл:

***

## Требования

| Компонент | Версия |
|-----------|--------|
| [Ollama](https://ollama.com) | ≥ 0.3 |
| Python | ≥ 3.10 |

***

## Установка и запуск

### 1. Установить Ollama

**Linux / macOS**

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

p.s. для макоси brew можно - `brew install ollama`

**Windows** — скачать установщик с [ollama.com/download](https://ollama.com/download)
и запустить `.exe`. - у меня нет винды, не проверял, но в интернете так прелагают

Проверить установку:

```bash
ollama --version
```

***

### 2. Скачать модель Qwen2.5:0.5B

```bash
ollama pull qwen2.5:0.5b
```

Убедиться, что модель загружена:

```bash
ollama list
# тут будет что-то типо : qwen2.5:0.5b   ...
```

***

### 3. Запустить Ollama-сервер

```bash
ollama serve
```

Сервер начнёт слушать на `http://localhost:11434`. Желательно, чтобы порт был свободен)

> **Примечание.** На Linux Ollama после установки часто запускается
> как systemd-сервис автоматически — команда `serve` в таком случае
> не нужна. Проверить статус: `systemctl status ollama`. (нашел такое в интернете, решил на всякий вставить, чтобы как в ТЗ все точно работало по гайду)

Проверяем, что сервер отвечает:

```bash
curl http://localhost:11434
# Должно вывестись: Ollama is running
```

***

### 4. Зависимости для скрипта

```bash
pip install requests
```

***

### 5. Запустить скрипт

```bash
python inference.py
```

По завершении в текущей папке появятся:

```
inference_report.html
```

Открываем `inference_report.html` через браузере, чтобы просмотреть отчёт.

***

## Ручная проверка через `curl`
можно еще руками запрос прокинуть
```bash
curl http://localhost:11434/api/generate \
  -H "Content-Type: application/json" \
  -d '{"model":"qwen2.5:0.5b","prompt":"Что такое рекурсия?","stream":false}'
```

Пример ответа:

```json
{
  "model": "qwen2.5:0.5b",
  "response": "Рекурсия — это рекурсия ...",
  "done": true
}
```

## Краткое описание функций скрипта

| Функция | Описание |
|---------|----------|
| `check_server()` | GET-проверка доступности Ollama-сервера |
| `generate()` | POST `/api/generate` → строка ответа модели |
| `run_inference()` | Прогоняет список запросов, возвращает результаты |
| `save_html()` | Генерирует HTML-отчёт с таблицей |
