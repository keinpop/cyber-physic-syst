# Выбор метрик

**Accuracy** — доля правильно классифицированных изображений, базовая метрика качества для задач классификации.  
**F1-score (macro)** — гармоническое среднее precision и recall, усреднённое по всем классам без учёта их весов; устойчива к дисбалансу классов и даёт честную оценку производительности по всем видам обезьян.  
**ROC-AUC (OvR, macro)** — площадь под ROC-кривой в схеме One-vs-Rest, усреднённая по классам; оценивает качество ранжирования модели независимо от порога классификации.

# Выводы

В рамках лабораторной работы была проведена полная реализация исследования компьютерного зрения для задачи многоклассовой классификации изображений 10 видов обезьян. Были успешно реализованы и сравнены предобученные модели torchvision (ResNet-18, ViT-B/16) и собственные имплементации (CustomCNN на базе ResNet-34, CustomViT на базе CCT).

**Ключевые выводы:**
1. **Предобученные модели значительно превосходят обучение с нуля** на малых датасетах (~1400 изображений). ViT-B/16 демонстрирует лучшие результаты (F1=0.9928) благодаря ImageNet-предобучению и архитектурным преимуществам трансформеров.

2. **Объединение train+val с random_split(80/20)** устраняет проблему нерепрезентативности оригинального val-set (~270 изображений), давая честную оценку качества.

3. **Собственные реализации требуют продвинутых техник:**
   - CustomCNN: переход от простых Conv-блоков к ResNet-34 с skip connections
   - CustomViT: замена patch embedding на ConvTokenizer (CCT) + SeqPool вместо CLS-токена

4. **Оптимизация критически важна:** AdamW + OneCycleLR + Label Smoothing позволяют собственным моделям достичь F1≈0.85–0.90, что составляет 85–90% от качества предобученных аналогов.

Работа демонстрирует понимание современных техник компьютерного зрения, включая transfer learning, knowledge distillation, архитектурные улучшения и правильную оценку моделей.
